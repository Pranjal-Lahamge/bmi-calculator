# 🧮 BMI Calculator

A simple and beginner-friendly BMI Calculator built using Python and Tkinter GUI.  
This project calculates Body Mass Index (BMI) based on user height and weight and shows the health category.

---

# 🚀 Features

- Calculate BMI instantly
- Clean and simple GUI
- Health category classification
- Beginner-friendly Python project
- Error handling for invalid input

---

# 🧠 BMI Formula

BMI = Weight (kg) / Height² (m²)

---

# 📂 Built With

- Python
- Tkinter GUI
